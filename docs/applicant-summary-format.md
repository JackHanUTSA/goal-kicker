# Applicant Summary Format

Goal:
Turn each university record into practical, source-backed guidance for a high school student.

Principles:
- never invent GPA ranges or requirements that are not in the record
- separate official requirements from competitiveness guidance
- surface uncertainty clearly
- keep the summary readable by a student, parent, or counselor

Required sections:

1. School overview
- school name
- rank if available
- majors count and count method
- record confidence

2. Academics needed
- testing policy
- GPA policy
- course-rigor notes
- short interpretation of what this means in practice

3. Essays and recommendations
- what written materials are required
- what recommendations are required
- any school-specific supplement notes

4. Projects / research / extracurricular signals
- direct project or research signals from the record
- leadership / extracurricular context if present
- practical interpretation for a student

5. Gaps / unknowns
- unknown fields
- warnings
- whether more verification is needed

Output style:
- concise but actionable
- use plain language
- prefer bullets over long paragraphs
- explicitly label uncertain or blocked fields

Example interpretation rules:
- if testing says "required" -> say the student should plan to test unless eligible for the stated exception
- if testing says "test-optional" -> say strong scores may still help when allowed, but do not claim they are required
- if course rigor mentions advanced or challenging courses -> say the student should pursue the strongest available curriculum in context
- if projects/research signals are sparse -> say the record does not strongly document project/research preference yet
- if unknown_fields is non-empty -> end with a caution note listing what remains unresolved
