# Goal Kicker Top 100 University Seed List

Expanded seed list for the Goal Kicker knowledgebase.

Pilot schools for first implementation and validation:
- Massachusetts Institute of Technology (MIT)
- Stanford University
- Harvard University
- University of California, Berkeley
- University of Michigan

Expanded target set is stored in:
- `data/top50_universities.json`

Notes:
- The filename remains `top50_universities.json` for backward compatibility with existing scripts.
- This is an operational seed list for the current build, now expanded to 100 schools.
- Ranking order should be treated as a seed/input artifact, not a claim of permanent truth.
- The agent should rely on official university domains for facts, not ranking pages for admissions requirements.
